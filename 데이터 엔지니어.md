# 데이터 엔지니어



- 하는 일
  - 데이터 **엔지니어링** 및 **플로우 자동화**
  - 데이터 **플로우 모니터링** 및 **성능 최적화**
  - 데이터 **추출 및 제공**
  - 대용량 데이터 효율적 사용을 위한 기반 데이터 **가공**





- 필요한 기술 스택
  - 프로그래밍 언어: Java, Python
  - 빅데이터 분석 플랫폼: Hadoop M/R, Hive, Spark 





- 우대사항





- Spark(중요 개념)
  - RDD(Resilient Distributed Dataset): 탄력적이며 분산된 데이터 셋
  - Lineage: 작업 플로우를 직접 실행하는 것이 아니라 작업 흐름에 대한 청사진을 작성.
